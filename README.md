## 🌕 Design
### 🌙 Concept
'문정해'에서 '문'은 영어로 Moon, 즉 '달'을 의미합니다. <br>
이름의 상징적인 의미를 고려하여 포트폴리오 홈페이지의 디자인 컨셉을 **달**로 설정했습니다. <br>
이를 바탕으로 특별하고 의미 있는 홈페이지를 제작하였습니다.
